# PlanetsAR

**PlanetsAR** is an augmented reality app that allows users to explore the solar system with realistic 3D models of planets. It provides an interactive and educational experience, bringing the wonders of space directly to your device.

## Features
- Explore detailed 3D models of planets in the solar system.
- Interactive Augmented Reality (AR) experience.
- Learn about the planets with rich information and visuals.
- Cross-platform support for iOS and Android devices.

## Installation

### Clone the repository
To get started, clone this repository to your local machine:
```bash
git clone https://github.com/yourusername/PlanetsAR.git
```

### Requirements
- Unity 2024 (or later) for AR development.
- AR Foundation and ARCore/ARKit plugins for cross-platform AR support.
- Xcode (for iOS development) or Android Studio (for Android development).

### Setting up the project
1. Open the project in Unity.
2. Make sure the required packages (AR Foundation, ARCore/ARKit) are installed via the Unity Package Manager.
3. Build and run the project on your device.

## Usage
1. Launch the app on your device.
2. Point your camera to an open surface, and the solar system will appear with interactive 3D models of the planets.
3. Tap on each planet to learn more about it.
